# Agents

<!-- https://agents.md -->

- This is a project to create UI sounds for Bootstrap.
- Every component which can be interacted with should have a sound.
- Sounds are off by default and require the data-ui-sounds attribute to be set (either on html element or body).
- Sounds can also be enabled by setting the data-ui-sounds attribute to true on a component.
- There should be a comprehensive test page in docs/index.html to test the sounds.

- The build script should generate a minified JS files.
- The script should be plain JavaScript and not use any frameworks or libraries.
- Use modern JavaScript features and syntax and web audio API.

## Components with sound

- [x] Alerts (when appearing)
- [x] Buttons (when clicking)
- [x] Carousel indicators (when clicking)
- [x] Collapse (when collapsing and expanding)
- [x] Dropdown
- [x] Form inputs
- [x] Modals
- [x] Popovers
- [x] Toast
- [x] Tooltips
- [x] Details / Summary
